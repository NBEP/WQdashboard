# Filtered site metadata (dataframe)

Dataframe containing site metadata.

## Usage

``` r
df_sites
```

## Format

A dataframe with 5-7 columns:

- Site_ID:

  Site ID

- Site_Name:

  Site name

- Latitude:

  Latitude

- Longitude:

  Longitude

- Town:

  Town

- State:

  State

- Watershed:

  Watershed name

- Group:

  Group. Values: Warmwater, Coldwater, Saltwater, NA
