# Site metadata (dataframe)

Dataframe containing raw site metadata.

## Usage

``` r
df_sites_all
```

## Format

A dataframe with at least 8 columns:

- Site_ID:

  Site ID

- Site_Name:

  Site name

- Latitude:

  Latitude

- Longitude:

  Longitude

- Town:

  Town

- State:

  State

- Watershed:

  Watershed name

- Group:

  Group. Values: Warmwater, Coldwater, Saltwater, NA

- Max_Surface_Depth_m:

  Maximum surface depth, meters. Default 1.

- Max_Midwater_Depth_m:

  Maximum midwater depth, meters

- Max_Depth_m:

  Site depth, meters

- Location_Type:

  Location type
