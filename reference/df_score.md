# Annual parameter scores (dataframe)

Dataframe containing annual parameter scores.

## Usage

``` r
df_score
```

## Format

A dataframe with 15-17 columns:

- Year:

  Year

- Site_Name:

  Site name

- Site_ID:

  Site ID

- Town:

  Town

- Watereshed:

  Watershed

- Group:

  Site group

- Depth:

  Depth category

- Parameter:

  Parameter

- Unit:

  Result unit

- score_typ:

  How annual score is calculated. Values: Minimum, Maximum, Average,
  Median, NA

- score_num:

  Numeric score

- score_str:

  Score. Values: Meets Criteria, Does Not Meet Criteria, Excellent,
  Good, Fair, Poor, No Threshold Established, No Data Available

- Longitude:

  Longitude

- Latitude:

  Latitude

- popup_loc:

  Popup text for map. Describes site.

- popup_score:

  Popup text for map. Describes parameter and score.

- alt:

  Alt text for map
