# Filtered result data (dataframe)

Dataframe containing filtered result data.

## Usage

``` r
df_data
```

## Format

A dataframe with 15 columns:

- Site_ID:

  Site ID

- Site_Name:

  Site name

- Watereshed:

  Watershed

- Date:

  Date

- Year:

  Year

- Parameter:

  Parameter

- Result:

  Result value

- Unit:

  Result unit

- Depth:

  Depth category

- Min:

  Minimum acceptable value

- Max:

  Maximum acceptable value

- Excellent:

  Threshold between excellent and good data

- Best:

  Whether excellent data is higher or lower than fair data

- Month:

  Month

- Description:

  Popup text for interactive graph
