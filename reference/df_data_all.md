# Result data (dataframe)

Dataframe containing raw result data.

## Usage

``` r
df_data_all
```

## Format

A dataframe with at least 13 columns:

- Site_ID:

  Site ID

- Activity_Type:

  Activity type

- Date:

  Date

- Depth:

  Sample depth

- Depth_Unit:

  Depth unit

- Depth_Category:

  Depth category

- Parameter:

  Parameter

- Result:

  Result value

- Result_Unit:

  Result unit

- Lower_Detection_Limit:

  Lower detection limit

- Upper_Detection_Limit:

  Upper detection limit

- Detection_Limit_Unit:

  Detection limit unit

- Qualifier:

  Qualifier code
